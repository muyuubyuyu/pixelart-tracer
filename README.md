# Pixel Art Tracer

A browser-based tool for tracing images into pixel art grids.  
Designed for creating layouts for Picross Touch and other grid-based puzzle games.

## Live Demo

Use directly in your browser:  
[muyuubyuyu.github.io/pixelart-tracer](https://muyuubyuyu.github.io/pixelart-tracer/)

---

## Features

- Selectable grid sizes from 5×5 up to 35×35  
- Upload an image and trace it with adjustable opacity  
- Lock background to prevent accidental movement  
- Paint and erase using the mouse  
- Mouse wheel zoom (when background is unlocked)  
- PNG export including grid and coordinate numbers  

---

## Controls

| Action | Result |
|-------|--------|
| Left click + drag | Paint pixels |
| Shift + drag | Erase pixels |
| Right click + drag | Move background image |
| Mouse wheel | Zoom background |
| Lock Background | Prevent moving and zooming the image |

---

## Purpose

This tool was built as a lightweight way to quickly design pixel layouts for logic puzzle games like Picross.
